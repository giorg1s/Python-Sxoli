# Student Management System
> **Σημείωση:** Αυτό το πρόγραμμα αποτελεί παράδειγμα χρήσης Αντικειμενοστρεφούς Προγραμματισμού σε Python. Επιδεικνύει τον ορισμό κλάσεων, την αρχικοποίηση αντικειμένων και τη διαχείριση δεδομένων μέσω μεθόδων.
> Είναι εργασία στα «Ολοκληρωμένα Περιβάλλοντα Ανάπτυξης (IDE) Ανοικτού Κώδικα» του Γ' Εξαμήνου απο τους: Φυλακούρης Γιώργος και Βεντουράτος Γιώργος-Λεωνίδας.
                
## 📋 Περιγραφή
Η εφαρμογή επιτρέπει τη δημιουργία οντοτήτων φοιτητών, την αποθήκευση βαθμολογιών και τον αυτόματο υπολογισμό του μέσου όρου τους. Είναι σχεδιασμένη για να τρέχει σε περιβάλλον VS Code με χρήση της επέκτασης Python.

## 🛠️ Τεχνικά Χαρακτηριστικά
* **Κλάση `Student`**: Περιλαμβάνει τις ιδιότητες `name`, `id_number` και τη λίστα `grades`.
* **Μέθοδος `add_grade`**: Επιτρέπει τη δυναμική εισαγωγή βαθμών στο αντικείμενο.
* **Μέθοδος `get_average`**: Υπολογίζει τον μέσο όρο, περιλαμβάνοντας έλεγχο για κενές λίστες ώστε να αποφεύγεται το σφάλμα `ZeroDivisionError`.

## 🚀 Οδηγίες Χρήσης
1. Αντιγράψτε τον κώδικα σε ένα αρχείο με κατάληξη `.py`.
2. Εκτελέστε το αρχείο πατώντας `F5` στο **Visual Studio Code**.
3. Τα αποτελέσματα θα εμφανιστούν στο ενσωματωμένο τερματικό.

## 💻 Παράδειγμα Κώδικα
Το πρόγραμμα δημιουργεί ένα αντικείμενο για τον φοιτητή "Γιώργο", προσθέτει δύο βαθμούς (18 και 20) και εκτυπώνει τον μέσο όρο (19.0).
